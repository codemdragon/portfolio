🛠 Installation & Setup

Get started with Codem Sidebar

🚀 Quick Start

Download the latest version from the Downloads section and run the executable. The sidebar will start automatically.

🔧 For Developers

If you want to integrate with the sidebar API from your own applications, install the Python client:

pip install requests

The sidebar client for API integration is now included in the main installation. You can find it in the installation directory.

🎯 First Run

When you first run the sidebar:

  • The sidebar will appear on your screen
  • Use the "Open Coord Grabber" button to capture screen coordinates
  • Create your first button with custom actions
  • Configure profiles for different workflows

🔐 API Overview

Control the sidebar programmatically

The Sidebar runs a local HTTP API at:

http://127.0.0.1:<port>
# Default: 8765

Authentication

All routes (except / and /info) require an API key:

Authorization: Bearer <your_api_key>

API Endpoints

Endpoint Method Description
/info GET Check if API is running
/profiles GET List all available profiles
/profiles/<profile>/buttons GET List buttons for a given profile
/profiles/<profile>/run/<button_id> POST Trigger button by numeric ID
/run POST Trigger button by label name
/run_temp POST Run temporary action sequence directly
/hidden_buttons GET List hidden buttons (Developer Mode)
/hidden_buttons POST Create hidden or temporary button

🧩 Quick Tutorials

Common API actions for integration

1

List all available Profiles

curl -H "Authorization: Bearer <api_key>" \
http://127.0.0.1:8765/profiles
2

List Buttons in a Profile

curl -H "Authorization: Bearer <api_key>" \
http://127.0.0.1:8765/profiles/default/buttons
3

Execute a Button by ID

curl -X POST \
-H "Authorization: Bearer <api_key>" \
http://127.0.0.1:8765/profiles/default/run/0
4

Execute a Button by Name

curl -X POST \
-H "Content-Type: application/json" \
-H "Authorization: Bearer <api_key>" \
-d '{"profile": "default", "label": "Open Chrome"}' \
http://127.0.0.1:8765/run
5

Run Temporary Actions

curl -X POST \
-H "Content-Type: application/json" \
-H "Authorization: Bearer <api_key>" \
-d '{"actions": [{"type": "move_rel", "value": "-100,0"}]}' \
http://127.0.0.1:8765/run_temp
6

Create Hidden Button

curl -X POST \
-H "Content-Type: application/json" \
-H "Authorization: Bearer <api_key>" \
-d '{"label": "MoveLeft", "actions": [{"type": "move_rel", "value": "-200,0"}]}' \
http://127.0.0.1:8765/hidden_buttons
